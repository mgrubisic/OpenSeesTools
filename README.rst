OpenSeesPy Extensions
+++++++++++++++++++++


Requirements
============

- Python 3.7+
- NumPy
- OpenSeesPy


Installation
============

.. code::

    pip install -e "git+https://github.com/otaithleigh/OpenSeesTools"
